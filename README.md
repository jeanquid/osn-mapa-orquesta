# 🎻 Orquesta Sinfónica Nacional — Mapa de Cargos

Mapa interactivo de los **105 cargos artísticos** de la Orquesta Sinfónica Nacional (OSN) según el Convenio Colectivo de Trabajo Sectorial, homologado por el **Decreto 669/2022**.

## Funcionalidades

- 🗺️ **Mapa orquestal interactivo** con disposición semicircular realista
- 🔍 **Buscador** de cargos por nombre (violín, trompeta, etc.)
- 🎨 **Filtros** combinables por Rol (Concertino / Solista / Suplente / Fila) y por Familia (Cuerdas / Maderas / Metales / Percusión)
- 📋 **Lista detallada** sincronizada con el mapa
- 📊 **Estadísticas** por rol: 2 Concertinos, 30 Solistas, 28 Suplentes, 45 de Fila

## Estructura de la OSN (Decreto 669/22)

| Rol | Categoría | Cantidad |
|-----|-----------|----------|
| Rol 1 | Concertinos | 2 |
| Rol 2 | Solistas | 30 |
| Rol 3 | Suplentes de Solista | 28 |
| Rol 4 | Fila (tutti) | 45 |
| **Total** | | **105** |

## Tecnologías

- React 18
- Vite 5
- Deploy: GitHub Pages

## Instalación local

```bash
git clone https://github.com/TU-USUARIO/osn-mapa-orquesta.git
cd osn-mapa-orquesta
npm install
npm run dev
```

## Deploy a GitHub Pages

```bash
npm run deploy
```

## Fuente

Convenio Colectivo de Trabajo Sectorial del Personal de Orquestas, Coros y Ballets Nacionales — [Decreto 669/2022 (DCTO-2022-669-APN-PTE)](https://www.argentina.gob.ar/normativa/nacional/decreto-669-2022-372428/texto)
